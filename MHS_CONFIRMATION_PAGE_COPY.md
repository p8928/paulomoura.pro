## Tecnologia aplicada em [domínio do cliente]

O site foi construído com Astro, um framework moderno voltado para sites rápidos, leves e bem estruturados. O Astro permite entregar muito conteúdo como HTML otimizado, reduzindo a quantidade de JavaScript enviada ao navegador. Para quem acessa o site, isso significa carregamento mais rápido, páginas que aparecem antes na tela e uma navegação que parece mais direta.

Nas partes que precisam de interatividade real, o projeto usa Svelte. O Svelte é uma tecnologia de interface que compila componentes de forma eficiente, evitando excesso de código no navegador. Em vez de transformar todo o site em uma aplicação pesada, [domínio do cliente] usa interatividade onde ela faz sentido: menus, formulários, seções dinâmicas, transições e componentes que respondem ao comportamento do usuário.

Para o usuário leigo, a vantagem é simples: o site comunica melhor. Elementos entram em cena no momento certo, a leitura ganha ritmo, a navegação fica mais fluida e a marca passa uma sensação de cuidado, solidez e precisão.

O uso de Lenis para rolagem suave, integrado ao GSAP, ajuda a criar uma experiência mais refinada sem perder controle técnico. O site também respeita preferências de redução de movimento, o que é importante para acessibilidade: pessoas sensíveis a animações intensas podem navegar com mais conforto.

Outros pontos técnicos relevantes reforçam o padrão High Standards: TypeScript para mais previsibilidade no desenvolvimento, Tailwind CSS para consistência visual e velocidade na construção da interface, fontes locais otimizadas, preload de recursos críticos, componentes reutilizáveis, testes com Vitest e Playwright, além de uma estrutura que separa bem conteúdo, layout, componentes e comportamento.
